# atividade-pratica-III

Contém 5 páginas desenvolvidas com HTML e CSS utilizando Flexbox.

Atividades
Atividade 1: Layout com menu lateral, cabeçalho e 3 blocos
Atividade 2: Página com hero, blocos, botão (CTA) e rodapé
Atividade 3: Dashboard com menu lateral e áreas de conteúdo
Atividade 4: Página em coluna com abas e conteúdo em 2 colunas
Atividade 5: Grade com 6 blocos (3 por linha)

Praticar o uso do Flexbox, utilizando propriedades como:

display: flex
flex-direction
flex
flex-wrap
gap
justify-content
align-items
